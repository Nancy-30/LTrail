# LTrail Frontend

React dashboard with ReactFlow visualization for LTrail traces.

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

The app will open at `http://localhost:3000`

## Features

- **Trace List**: Sidebar showing all traces with status indicators
- **Flow Visualization**: ReactFlow diagram showing step-by-step decision trail
- **Real-time Updates**: WebSocket connection for live trace updates
- **Step Details**: Detailed view of step inputs, outputs, and evaluations
- **Tailwind CSS**: Modern, responsive UI

## Build for Production

```bash
npm run build
```

## Technologies

- React 18
- ReactFlow 11
- Tailwind CSS 3
- Axios
